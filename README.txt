Organization Name: GlassIntact
Product Name: GlassIntact


A cup that has an embedded sensor that can detect if a drink has been 
spiked or drugged. Through the use of visual cues in the form of an LED 
light at the bottom of the glass, consumers can be alerted when their 
drink has been tampered with. 

Github pages link: https://glassintact.github.io/GlassIntact/
